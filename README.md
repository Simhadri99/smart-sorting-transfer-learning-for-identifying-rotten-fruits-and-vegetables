# Smart Sorting: Rotten Fruits & Vegetables Detection

## Project Overview
This project uses Transfer Learning with MobileNetV2 to classify fruits and vegetables as Fresh or Rotten.

## Technologies Used
- Python
- TensorFlow
- MobileNetV2
- Transfer Learning

## How to Run

1. Install requirements:
pip install -r requirements.txt

2. Train model:
python train.py

3. Test:
python app.py
